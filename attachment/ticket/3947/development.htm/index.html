Development Center

		
The
jQuery UI Team is proud to be part of a vital and active community of
developers. We encourage everyone who's interested to jump in and
participate in the development of the library. We aim to quickly fix
bugs, implement new features, and grow the jQuery UI library into an
innovative, useful, and accessible resource for developers of all
skill-levels.
 
	

	
	
		
		
Development Status
 
		
Last updated February 27, 2009 at 6:45AM EST

		
  
The final release candidate (1.6rc6)
is out and being tested now. Based on feedback from the jQuery UI
community, we're going address the confusion around compatibility
between jQuery 1.2.6 and 1.3 and the jQuery UI library by creating two
very distinct releases in the next few weeks:


 
What we are
currently calling jQuery UI 1.6rc6 is going to be released as jQuery UI
1.7. This code is built from the ground up to take full advantage of
jQuery 1.3. The 1.7 release will include major and minor fixes and
updates to each 1.5 component, the new Progressbar plugin, new jQuery UI CSS Framework and ThemeRoller v2.0 app.
 


For all those folks still actively using jQuery 1.2.6, we want to
provide a legacy release of the jQuery UI library based on 1.6rc2 that
ports over as many bug fixes and improvements as possible. To avoid any
confusion, this version will be called jQuery UI 1.6 final. This
release will have the same theming support as jQuery UI 1.5.3.
 


 To learn more about the dual release plan, please read the jQuery 1.7 is the new 1.6  blog post. To see the latest updates on our development progress and plans for the future, please visit the jQuery UI Development & Planning Wiki
