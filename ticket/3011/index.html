Interactions should support bottom and right positioning

Reported by: scottgonzalez Owned by:
Priority: major Milestone: 2.0.0
Component: ui.mouse Version: 1.5
Keywords: Cc:
Blocked by: Blocking:
Description

Resizable and draggable should support bottom and right positioning in addition to the existing top and left positioning.

If it's not possible to detect if the element is already positioned by the bottom or right, then an option should be added for which two sides to use for positioning (default to top left).
