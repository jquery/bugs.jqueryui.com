Updates to Gallery Themes

Reported by: fgTodd Owned by:
Priority: major Milestone: 1.7.2
Component: [meta] ui.themeroller Version: 1.7.1
Keywords: Cc:
Blocked by: Blocking:
Description

7 New themes added to the gallery: Sunny, Overcast, Le Frog, Flick, Pepper Grinder, Eggplant and Dark Hive. This brings to total number of gallery themes to 23.

5 Current themes were renovated to clean up the design and add visual polish:

Blitzer - Adjusted header stroke color, standardized base font size to 1.1 em to match other themes

South Street - Header color now a warm light gray, green colors throughout tweaked to be a bit brighter, shadow and modal div styles adjusted, standardized base font size to 1.1 em to match other themes

Cupertino - blue color throughout is a bit brighter and all light grays are now light blue, standardized base font size to 1.1 em to match other themes

Mint Choc - Overall the theme is a darker and richer to improve contrast and increase chocolaty goodness.

Excite Bike - Standardized base font size to 1.1 em to match other themes

*** Once all these changes are released, we need to update Google's CDN and the theme switcher to include these new themes with CDN links. ***
