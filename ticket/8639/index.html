jQuery 1.9.0 dialog on a XHTML page is broken if buttons are defined

Reported by: at4u Owned by:
Priority: minor Milestone: 1.9.1
Component: ui.dialog Version: 1.9.0
Keywords: Cc:
Blocked by: Blocking:
Description

I just tried to use 1.9.0 on a website using XHTML 5. If I define a button to be shown on a dialog not all browsers but Firefox reports (translated):

Error: No corresponding tag. Expected: </button>.
Source: <div xmlns="http://www.w3.org/1999/xhtml"><button type='button'></div>

Furthermore no dialog is displayed.

Example "demos/dialog/modal-message.html" changed to "demos/dialog/modal-message.xhtml":
