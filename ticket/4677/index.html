Description

Most of us who are new to JQuery were expecting not to know much of CSS before solving issues like z-index. I was able to finally figure out how to set z-index using firebug and eventually ended up setting the z-index on my base/ui.datepicker.css class

As I was not sure where to see what my asp.net master pages and all those theme.css classes were doing (this is just within the .NET world
