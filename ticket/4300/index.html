Cupertino theme differs 1.5.3 vs. 1.6/1.7

Reported by: pbuck Owned by:
Priority: minor Milestone: 1.6
Component: [meta] ui.themeroller Version: 1.6rc2.5
Keywords: Cc:
Blocked by: Blocking:
Description

The Cupertino theme generated for UI 1.5.3 via themeroller is generally grey. The

Cupertino theme generated for UI 1.6/1.7 has the blue highlights which are

also present in the online example (light blue dialog box header is

simple obvious difference.) The newer version has what appears to be the correct/preferable configuration.

1.7 dialog buttons float right, 1.5.3 don't. There are (likely) more differences, but these two are immediately obvious. I don't have any information regarding differences with other themes.
