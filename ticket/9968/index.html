The intention of the delay option is to prevent accidental drags, so it doesn't make sense that the user is intentionally waiting for the delay to end to be notified that they can drag the item. If you really want to do this, you can just use .one( "sort", ... ) inside start and unbind it in stop just in case the sort never occurs.

The delay option is being removed as part of the rewrite (see more at http://wiki.jqueryui.com/Sortable). If you need more help, please use the forums or Stack Overflow.
