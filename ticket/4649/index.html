Ticket-specific RSS feed link is missing domain

Reported by: DarwinSurvivor Owned by:
Priority: trivial Milestone: 1.8.6
Component: [meta] ui.site Version: 1.7.2
Keywords: RSS Cc:
Blocked by: Blocking:
Description

The RSS feed for a ticket is displayed as:

/ticket/4648?format=rss

instead of

http://dev.jqueryui.com/ticket/4648?format=rss

in the HTML code of the website.

This causes firefox to attempt to "download" the xml file (since it has not domain name) instead of subscribing to it.

Seems like a simple fix for a "papercut" level bug.
