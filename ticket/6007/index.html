Description

I just updated jqueryui to 1.8.4 and I return JSON data back as a result. My label contains HTML however, autocomplete doesn't parse html properly. It just displays HTML as string. It wasn't the case in 1.8.2

For example my result json string is :

Instead of parsing the HTML it shows the HTML text in the list. In 1.8.2, the HTML is properly rendered.
